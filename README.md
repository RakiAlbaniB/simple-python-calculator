# simple-python-calculator

# Python Calculator GUI
Kalkulator sederhana yang dibangun menggunakan library Tkinter dengan fitur matematika dasar, akar kuadrat, dan pangkat.

## Fitur
- Operasi Dasar (+, -, x, /)
- Sisa Bagi (%)
- Pangkat 2 dan Akar Kuadrat
- Error Handling (Zero Division)

## Cara Menjalankan
1. Clone repositori ini
2. Jalankan perintah: `python calculator.py`
